# Asset licenses

All third-party game assets in `apps/client/public/assets/kenney` are from
[Kenney](https://kenney.nl/) and released under CC0 1.0.

| Pack | Source | Usage |
| --- | --- | --- |
| Top-down Shooter | https://kenney.nl/assets/top-down-shooter | Characters, zombies, weapons, arena tiles and props |
| UI Pack | https://kenney.nl/assets/ui-pack | UI font and interface textures |
| Impact Sounds | https://kenney.nl/assets/impact-sounds | Hit and environment feedback |
| Interface Sounds | https://kenney.nl/assets/interface-sounds | Menu and wave feedback |

The original `License.txt` files are preserved beside each asset group.

Generated concept art is design reference only and is not shipped as gameplay
content.

## Original generated gameplay assets

The files in `apps/client/public/assets/custom` were generated specifically for
Duo Outbreak with OpenAI ImageGen, then locally chroma-keyed and optimized:

- Spitter ranged mutant
- Spitter bio-energy projectile
- Medical case pickup
- Armor plate pickup
- Weapon unlock crate
- Original green mutation combat form

They are original project assets and do not reproduce Boxhead artwork.
